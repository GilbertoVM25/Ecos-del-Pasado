# Ecos-del-Pasado
<img width="256" height="256" alt="Ecos Logo" src="https://github.com/user-attachments/assets/f828e5d0-a724-41b9-9458-991f7b0ceec9" />

Juego de plataformas y acción para celulares inspirado en la historia y cultura prehispánica hecho en Unity con lenguaje C#.

## Características
- Plataformas 2D
- Sistema de diálogos
- Guardado de progreso
- Enciclopedia desbloqueable
- Localización de contenido

## Tecnologías
- Unity
- C#
