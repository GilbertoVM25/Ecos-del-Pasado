# Ecos-del-Pasado
Juego de plataformas y acción para celulares inspirado en la historia y cultura prehispánica.

## Características
- Plataformas 2D
- Sistema de diálogos
- Guardado de progreso
- Enciclopedia desbloqueable
- Localización de contenido

## Tecnologías
- Unity
- C#
- Firebase
- Git
